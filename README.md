# SnowMountainWar
 
使用Unity開發的一款3D雙人對戰遊戲，玩家們在雪山村莊中，一邊躲藏在村落之中，一邊射擊雪球攻擊對手來削減生命值，將對手生命值歸零即可獲勝。

Unity版本: 2020.3.8f1

## 遊戲操作
|  | Player1 | Player2 |
| -------- | -------- | -------- |
| 前進    | W     | ↑     |
| 後退    | S     | ↓     |
| 左轉    | A     | ←     |
| 右轉    | D     | →     |
| 發射    | Space | 0    |
| 結束遊戲 |ESC          |
